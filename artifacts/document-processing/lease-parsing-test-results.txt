ERROR ANALYSIS
═══════════════════════════════════════════════════════════════════════════════

Common Extraction Failures:

1. Lease End Date (2 failures - 10% error rate)
   Reason: Non-standard date formats ("end of month", "12 months")
   Impact: Medium
   Remediation: Add fuzzy date parsing for relative dates

2. Tenant Name (1 failure - 5% error rate)
   Reason: Multiple tenants listed, parser extracted only first
   Impact: Low
   Remediation: Parse comma-separated tenant names

3. Utilities Included (8 failures - 40% error rate)
   Reason: Utilities mentioned but unclear if included in rent
   Impact: Low (optional field)
   Remediation: Improved context analysis

4. Parking Spaces (9 failures - 45% error rate)
   Reason: Parking mentioned separately or in addendum
   Impact: Low (optional field)
   Remediation: Parse addendums and attachments

5. Square Feet (6 failures - 30% error rate)
   Reason: Abbreviation variations ("sf", "sq.ft.", "square feet")
   Impact: Low (optional field)
   Remediation: Add more pattern variations
